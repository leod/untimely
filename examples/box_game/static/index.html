Untimely: Box Game

Example 1: Local Gameplay

The game runs locally only -- no multiplayer.

Controls: W/A/S/D for moving the green box around.

Example 2: Client-Server

The server periodically sends out the game state to all clients. Clients are "stupid terminals": they simply display the state that they received as soon as they get it. In return, clients send their game input to the server.
